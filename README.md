# Spotify_Final_Project_H501
This project is part of the INFO-H501: Introduction to Data Science Programming course at Indiana University.  
It explores Spotify song data through visualizations and an interactive web app built using Streamlit.

## Features
- Interactive visualization for comparing up to three audio features (such as energy, danceability, valence) across genres  
- Custom feature selection and dynamic charts  
- Data exploration of average values and track popularity  
- Simple, user-friendly design built in Streamlit

## Files
- spotify_features_app.py – Streamlit app for interactive visualization  
- spotify_songs.csv – Dataset used for analysis  
- Spotify_Project_Calculations_Practice.ipynb – Testing and analysis notebook  
- Testing_Project_Mahya.ipynb – Individual experimentation notebook
